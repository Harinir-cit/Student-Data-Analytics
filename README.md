This project analyzes student academic performance using Python.

- Department
- CGPA
- Attendance
- Marks
- Placement
- Internship

## Tools Used

- Pandas
- Matplotlib
- Seaborn

## Analysis
- Average CGPA
- Attendance Analysis
- Placement Analysis
- Department-wise Performance
